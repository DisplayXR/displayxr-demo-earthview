EarthView

A real-time streaming 3D city viewer for glasses-free 3D displays, built on the DisplayXR OpenXR runtime via Vulkan. EarthView streams Google Photorealistic 3D Tiles — fly the full-scale world maps-style, or double-click to frame a neighborhood as a tabletop diorama inside the display's depth volume, rendered with asymmetric per-eye Kooima projection.

Once installed, the app lands at /Applications/EarthView.app. Cycle preset cities with B (Paris, San Francisco, New York, Tokyo, Sydney); double-click to acquire an orbit center, C to fly again.

Google Map Tiles API key required. EarthView needs a key to stream tiles: set GOOGLE_MAPS_API_KEY or put key=… in earthview.ini next to the app. Enable the Map Tiles API in the Google Cloud Console and create a key. Without one, the app opens to a how-to card and does not crash. The key is never bundled.
Prerequisite: the DisplayXR runtime must be installed first. Download DisplayXR-Installer-*.pkg from the displayxr-runtime releases page. The demo bundles its own copy of MoltenVK and the OpenXR loader, but it discovers the actual runtime through the system's OpenXR configuration registered by the runtime installer.
